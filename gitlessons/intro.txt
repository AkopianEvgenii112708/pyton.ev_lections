GIT - распределенная система контроля версий. Эта система для отслеживания и ведения исории изменения файлов в вашем проекте.

Репозиторий - хранилище вашего кода и истрии его изменений.

GitHub - Веб сайт для размещения git-репозиториев в совместной разработке проектов.

Команды Git:

1) git init - Создает новый локальный гит репозиторий у вас на пк, в той директории в которой была прописана команда. (вводится один раз в начале)
2) git add - команда для добавления файлов в систему отслеживания Git, после этой команды изменения начинают отслеживаться гитом.
3) git commit -m 'comment' - как только достигаем определенного момента разработки, сохраняем и коммментируем изменения в локальный репозиторий. 
4) git remote add - связывает наш локальный репозиторий с удаленным. То есть добавляет подключение к репозиторию на гитхабе.  (git remote add <name: origin> <ссылка на репозиторий ssh>)
5) git push - команда для отправки всех изменений на локальном репозитории в удаленный репозиторий (get push origin <main>)
